# ☁️ Cloudflare Workers Setup

## Architecture

- **Backend API**: Cloudflare Workers
- **Database**: Firebase Firestore (stays the same)
- **Frontend**: Already deployed to Firebase Hosting (or can move to Cloudflare Pages)

## Steps

1. Install Wrangler CLI
2. Create Cloudflare Workers project
3. Adapt code for Workers environment
4. Deploy to Cloudflare

Let's start!
